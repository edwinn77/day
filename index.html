📖 择吉算法说明

☀️ 黄道吉日（青龙起例）

十二星神按固定顺序循环，其中六位为黄道吉神：

青龙 ✓ 明堂 ✓ 天刑 朱雀 金匮 ✓ 天德 ✓ 白虎 玉堂 ✓ 天牢 玄武 司命 ✓ 勾陈

每月"青龙"的起始日支不同：

干支月青龙起于
寅月/申月子日
卯月/酉月寅日
辰月/戌月辰日
巳月/亥月午日
午月/子月申日
未月/丑月戌日

🌟 九星吉日

六十甲子排成九列，每列一颗星。其中 煞贡、直星、人专 为吉星：

妖星 惑星 禾刀 煞贡 ✓ 直星 ✓ 卜木 角己 人专 ✓ 立早

按月份组别（孟月/仲月/季月）确定哪些余数为吉。

📐 建除十二直

日支与月支的关系决定十二直。其中 成、开、定 日为大吉，破、闭 日为凶。每日具体宜忌由十二直决定。

🚫 排除条件

即使九星属吉，还需排除：

金神七煞（年禁忌）
甲/己年忌午未 · 乙/庚年忌辰巳 · 丙/辛年忌子丑寅卯 · 丁/壬年忌戌亥 · 戊/癸年忌申酉
红煞日（月禁忌）
孟月忌酉日 · 仲月忌巳日 · 季月忌丑日

🔨 大偷修日

以下八日特别适合修造：
壬子 · 癸丑 · 丙辰 · 丁巳 · 戊午 · 己未 · 庚申 · 辛酉

✅ 综合评估公式

综合评分 = 黄道吉神 + 建除十二直 + 九星吉值 − 年/月大煞

三套系统叠加评估，得出上上吉→大凶六个等级，给出最终择吉建议。
